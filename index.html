Sobre

Alex Ricardo

Desenvolvedor Web | Desenvolvedor Mobile


Meu nome é Alex Ricardo e eu desenvolvo sites e aplicativos. Sou um víciado e apaixonado por códigos! Eu venho projetando e codificando na web desde 2017.

Minha primeira experiência com HTML, CSS, JavaScript e PHP apareceu quando eu estudava no Instituto Federal de educação, ciência e tecnologia de Pernambuco(IFPE) e estava cursando Técnico em informática para Internet.

No decorrer da minha carreira de estudante, descobrir novas maneiras de desenvolver funcionalidades na web utilizando HTML, CSS e JavaScript. Eventualmente, isso me levou a construir sites melhores, mais rápidos e bonitos.

Aprimorei meus conhecimentos aprendendo novas bibliotecas e frameworks para auxiliar nas criações dos websites, até então construir alguns projetos para web e os mesmos podem ser encontrados no meu GitHub.

Algumas tecnologias que tenho conhecimento:

Front-end: HTML5 • CSS3 • JAVASCRIPT
Back-end:  PHP7 • JAVA • SQL • NODE.JS
Frameworks/Bibliotecas: Bootstrap 4 • Materialize • Jquery • React Native • Material Design

Abaixo contêm alguns sistemas/Landing Pages/Aplicativos que já fiz

    1. Workfast
    2. SystemAcad
    3. GreenLeaf

O Workfast é um site voltado para interligar os prestadores de serviços com os clientes, o qual tem intuito de tentar amenizar a falta de empregos em nosso país. Esta plataforma facilita aos candidatos(fornecedores de serviços) a vantagem de cadastrarem seus serviços de forma gratuita, contemplando profissionais de todos os perfis.

Tecnologias utilizadas para construção do WebSite:

  • •   HTML 5
  • •   CSS 3 (Bootstrap 4)
  • •    JavaScript (Jquery • SweetAlert • Ajax • PDF JS • API Google Maps)
  • •     JAVA (Hibernate, Maven e Spring-MVC )
  • •    SQL

O SystemAcad é um site acadêmico que tem por finalidade auxiliar instituições que queiram administrar melhor seus alunos, professores e funcionários.

Tecnologias utilizadas para construção do WebSite:

  • •   HTML 5
  • •   CSS 3 (Bootstrap 4)
  • •    JavaScript (Jquery • Ajax)
  • •     PHP
  • •    SQL

O projeto GreenLeaf é um site que passou a tomar forma a partir do interesse de jovens estudantes sobre sustentabilidade, porém não apenas isso, mas também o desejo de ajudar e guiar pessoas a terem uma visão diferenciada sobre a ecológia.

Tecnologias utilizadas para construção do WebSite:

  • •   HTML 5
  • •   CSS 3 (Bootstrap 4)
  • •    JavaScript
  • •     PHP
  • •    SQL
